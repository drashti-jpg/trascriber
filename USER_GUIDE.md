## ğŸ’¡ **Pro Tips**

### **Organization**
1. **Create project per client/team** - Easy to find all related meetings
2. **Use consistent naming** - "Weekly Sync - [Date]" pattern
3. **Pick meaningful colors** - Visual identification at a glance
4. **Fill all meeting details** - Better searchability later

### **Recording Quality**
1. **Test first** - Do 10-second test before important meetings
2. **Use headphones** - Prevents echo and feedback
3. **Stable internet** - Better quality for online meetings
4. **Quiet environment** - Reduces background noise
5. **Check "Share audio"** - #1 missed step for tab/system audio!

### **Format Selection**
1. **Internal MOM** - Quick team updates, standup notes
2. **Client MOM** - Professional client communications
3. **Standard** - When you need exact wording preserved
4. **Always fill attendees** - Helpful for future reference

### **After Saving**
1. **Review formatted output** - Check it meets your needs
2. **Edit if needed** - Copy and edit in external editor
3. **Share immediately** - Copy and paste into email/Slack
4. **Archive important ones** - Download to local storage

## ğŸ”§ **Troubleshooting**

### **"No audio track found" Error**
**Fix:** You forgot to check "Share audio" checkbox!
- Try again and carefully check the box

### **Can't hear my own voice**
**Fix:** Make sure you allowed microphone access
- Browser Tab/System Audio needs BOTH permissions

### **Format modal doesn't appear**
**Fix:** Transcription may have failed
- Check error messages
- Verify API key is configured

### **Can't create new project**
**Fix:** Project name is required
- Must enter at least a name

### **Transcript not saving**
**Fix:** Meeting title is required
- Must enter a title to save
